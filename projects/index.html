Lighting and Materials in WebGL

The goal of this project was to experiment with different lighting styles and materials by hand-coding shaders that implement different lighting calculation techniques with adjustable material properties.

The scene consists of a number of different objects. In the foreground are a hand-coded cube and sphere along with a teapot and bunny which all slowly rotate. In the background are a series of animated connected assemblies featuring: a set of square pyramids rotating about each other and a rotating together along the red track, a recursively drawn tree, and a "wagging" tail. Finally, there is also a ground plane and set of world coordinate axes.

You can use the WASD keys to control the camera's position and click and drag the mouse to orient it. There are several drop down menus, sliders, and color-pickers that adjust various properties of the scene.

Calculating the Distance to the LMC

Calculating distance in Astronomy is difficult and it gets even more difficult the farther things are. For objects close to us, we can use parallax to measure distance. It is relatively simple geometric method for calculating distance and the GAIA satallite pushes this method to its limit using the entire orbit of the Earth as its baseline. But even then, it is only reliable out to a few thousand light years. For distances much further, astronomers employ the cosmic distance ladder which is a series of measurements that build on each other to measure further and further distances. One of the earlier rungs are Cepheid Variable (CV) stars which puslate with a measureable period-luminosity relationship.

The goal of this project was to determine the period-luminosity relationship for Cepheid Variable stars and then use that relationship to determine the distance to one of our closest galactic neighbors, the Large Magellenic Cloud (LMC).

Temporal luminosity data for the CVs was collected from OGLE and the Lomb-Scargle Periodogram was used to determine the period of the CVs within the galactic bulge. The CVs from OGLE were cross-referenced with GAIA data to get their parallax measurements. This data was combined with extinction data from IRSA to determine the period-luminosity relationship. The calculated linear relationship was then used on CV data from the LMC to determine the distance using the distance modulus.

Finding Bigfoot

This is a data visualization project based off data collected by the Bigfoot Field Researchers Organization (BFRO) which is available for download here. This was group project completed by Ben Fisk, Jamie Lee, Yousef Farge, and myself. The goal of this project was to make a convincing data visualization to explore Bigfoot sightings and allow the user to determine if the legend of Bigfoot holds any merit or is pure fiction.

The project consists of two primary connected elements: a trend plot and a map. The trend plot shows three seperate lines for each class of observation and has interactive popups for historic Bigfoot-related events that may have influenced reported sightings and a brush that allows users to select a time range of reports to display on the map. The map has a single point for every reported sighting within the selected region on the trendplot. Clicking on a state shows more information about sighitngs within that state and a drop down menu allows the user to add a chloropleth scale to the map displaying one of several different statistics for each state.
